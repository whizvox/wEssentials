# wEssentials

This is a small Minecraft Paper plugin to give servers many essential features, such as teleport requests (`/tpa` and
`/tphere`), warp points (`/warp`), kits (`/kit`), player nicknames (`/nick`), and chat customization.

Very work-in-progress, don't use yet!
